# DoubleVisionJS
They've successfully create a portal to a parallel dimension,
but we've unknowingly unleashed a terrifying hoard of monsters!

Fight the horde for as long as possible!

## Controls 

### Left Guard
* W - Up
* S - Down
* D - Fire

### Right Guard
* UP - Up
* DOWN - Down
* LEFT - Fire
